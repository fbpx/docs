# Input

Shows the input defined in the flow file.

e.g.
```
$ fbpx input test.fbp

 Input Data for Example flow

┌───────────────────────┬──────┬─────────┐
│ Data                  │ Port │ Process │
├───────────────────────┼──────┼─────────┤
│ "http://example.com/" │ url  │ Request │
└───────────────────────┴──────┴─────────┘
```
